# 3D 骰子投擲模擬器

使用 Three.js 和 Cannon.js 物理引擎實現的互動式 3D 骰子投擲應用程式。

## ✨ 功能特色

-   🎯 **真實物理模擬**：使用 Cannon.js 物理引擎實現真實的骰子投擲效果
-   🎨 **精美 3D 視覺**：基於 Three.js 的高品質 3D 渲染
-   🎮 **互動控制面板**：
    -   調整骰子數量（1-10 個）
    -   調整投擲力度（5-30）
    -   調整投擲高度（2-10）
-   📊 **即時統計**：自動計算並顯示每個骰子的點數和總點數
-   🌈 **彩色骰子**：每個面都有不同的顏色，易於辨識
-   🔄 **重置功能**：一鍵清除所有骰子

## 🚀 快速開始

### 方法 1：直接開啟（推薦）

1.  下載所有檔案到同一個資料夾
2.  直接在瀏覽器中開啟 `index.html`
3.  開始投擲骰子！

### 方法 2：使用本地伺服器

```bash
# 使用 Python 3
python -m http.server 8000

# 使用 Python 2
python -m SimpleHTTPServer 8000

# 使用 Node.js (需要先安裝 http-server)
npx http-server
```

然後在瀏覽器中訪問 `http://localhost:8000`

## 🎮 使用說明

1.  **調整參數**：
    -   使用滑桿調整骰子數量、投擲力度和高度
    -   即時顯示當前設定值
2.  **投擲骰子**：
    -   點擊「投擲骰子 🎲」按鈕
    -   骰子會從設定的高度落下
    -   等待 3 秒後自動顯示結果
3.  **查看結果**：
    -   每個骰子的點數會顯示在統計區域
    -   自動計算總點數
4.  **重置場景**：
    -   點擊「重置場景 🔄」按鈕清除所有骰子

## 🛠️ 技術棧

-   **Three.js (r128)**：3D 圖形渲染
-   **Cannon.js (0.6.2)**：物理引擎
-   **原生 JavaScript**：應用邏輯
-   **CSS3**：樣式和動畫

## 🎨 特色功能詳解

-   **物理模擬**
    -   重力加速度：-30 m/s²
    -   線性阻尼：0.3
    -   角度阻尼：0.3
    -   碰撞檢測：邊界牆和地面
-   **視覺效果**
    -   環境光 + 方向光 + 點光源
    -   即時陰影渲染
    -   霧效果增強景深
    -   漸變背景色
-   **骰子設計**
    -   6 個面，每面不同顏色
    -   （註：此版本為簡化版，未實現點數貼圖，僅透過向量計算點數）
    -   真實的點數分布（1-6）
-   **自動點數檢測**

## 🔧 自訂設定

您可以在 `app.js` 中修改以下參數：

```javascript
// 重力
world.gravity.set(0, -30, 0);

// 骰子大小
const size = 1;

// 場景顏色
scene.background = new THREE.Color(0x1a1a2e);

// 骰子顏色
const colors = [0xff6b6b, 0x4ecdc4, 0x45b7d1, 0xf7b731, 0x5f27cd, 0x00d2d3];
```
